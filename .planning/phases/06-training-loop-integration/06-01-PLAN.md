Create `examples/train_simple.rs` — a working training loop.

Architecture:
- 2-layer model: conv1x1(64->64) -> conv1x1(64->64)
- Input: random fp32 tensor [1, 64, 1, 16]
- Target: random fp32 tensor [1, 64, 1, 16]
- Loss: MSE(output, target)
- Gradient: computed on CPU using MSE gradient formula
- Optimizer: SGD with lr=0.01
- Weight update: CPU, then delta compilation reload

Implementation:
1. Initialize ANE, create DeltaCompiler
2. Compile 2 conv1x1 layers, store weights as Vec&lt;f32&gt;
3. Generate synthetic input/target pairs
4. Training loop (50 steps):
   a. Forward layer 0: ANE eval
   b. Forward layer 1: ANE eval (input = layer 0 output)
   c. Loss: MSE(final_output, target)
   d. Gradient: dL/d(out) = 2*(output - target)/N
   e. grad_W1 = dL * h0^T (outer product, CPU)
   f. dL/dh0 = W1^T * dL (matmul, CPU), grad_W0 = dL_dh0 * x^T
   g. SGD: W -= lr * grad_W
   h. Delta compile: reload both layers
   i. Print step, loss, time
5. Assert: loss decreases from step 0 to step 49

Use small values for weights (0.01-0.5) to stay in fp16 safe range.
Print "OK train_simple" on success.
  </action>
