# Existing infrastructure:
# - ANECompiler::compile_multi() handles multiple weight files
# - ANEExecutor::reload_weights() takes &[(&str, &[u8])] pairs
# - WeightBlob::from_f32() converts FP32 → FP16 blob
# - fwd_ffn_mil(seq, dim, hidden) generates a valid ANE MIL program
# - examples/delta_compilation.rs shows the basic pattern

# Key insight from Orion paper:
# Delta compilation = reload_weights() which patches weight blobs without recompiling
# This is ~8.5x faster than full recompilation (~494ms vs ~4200ms)

# For multi-layer: compile 4 separate single-layer programs, each with its own weights
# Then reload all 4 in a loop, measuring total reload time
</context>
