   - Assert: no errors in any cycle

7. Run selective update test:
   - Reload weights for layer 1 ONLY (not layer 0 or 2)
   - Eval layer 0 with same input → output should match previous cycle
   - Eval layer 1 → output should change
8. Print budget status: compile_count, remaining, warning status
